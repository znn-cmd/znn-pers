# 🔲 Обновление логотипа ZNN - Квадратные углы

## ✅ Изменение выполнено

Убраны скругленные углы у логотипа ZNN - теперь он имеет строго квадратную форму.

---

## 🔄 Что изменилось

### До:
```css
rounded-lg    /* Header - 6px скругление */
rounded-xl    /* About - 12px скругление */  
rounded-lg    /* Contact - 6px скругление */
rounded-2xl   /* Home - 16px скругление */
rounded-2xl   /* Card - 16px скругление */
```

### После:
```css
/* Убраны все rounded классы */
/* Теперь строго квадратные углы */
```
