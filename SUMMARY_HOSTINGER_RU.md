## Структура развертывания на VPS

```
Интернет
    ↓
    │ HTTPS (443)
    ↓
  Nginx (Обратный прокси)
    │ - SSL терминация
    │ - Кэширование
    │ - Сжатие
    ↓
    │ HTTP (3000)
    ↓
  Next.js приложение
    │ Управляется PM2
    │ - Авто-рестарт
    │ - Мониторинг
    │ - Логи
    ↓
  Файловая система
    │ /var/www/znn
    │ - Код приложения
    │ - MDX контент
    │ - Статические файлы
```

## Технические детали

### Сборка
```bash
npm run build
```
Генерирует:
- 100 HTML страниц
- Оптимизированный JavaScript (86.2 kB shared)
- Статические ассеты
- Standalone сервер

### Файловая структура после сборки
```
.next/
├── standalone/          # Минимальный сервер
├── static/             # Статические файлы
└── server/             # Серверные компоненты

.next/standalone/
├── server.js           # Entry point
├── node_modules/       # Только необходимые зависимости
└── .next/
    ├── pages/
    └── ...
```

### Требования к серверу

**Минимум:**
- CPU: 1 vCore
- RAM: 2GB
- Диск: 10GB
- Node.js: 18+

**Рекомендуется:**
- CPU: 1-2 vCore
- RAM: 4GB
- Диск: 20GB
- Node.js: 18 LTS

**Hostinger VPS 1 идеально подходит** ✅
